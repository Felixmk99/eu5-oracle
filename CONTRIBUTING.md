# Contributing to EU5 Oracle

Thank you for your interest in contributing to the EU5 Oracle! We welcome contributions to make this strategic advisor even better.

## How to Contribute

### 1. Reporting Bugs
- Open an issue on GitHub describing the bug.
- Include steps to reproduce the issue.
- If possible, provide logs or screenshots.

### 2. Suggesting Features
- Open an issue to discuss your idea before implementing it.
- Explain the "why" and "how" of your feature.

### 3. Submitting Changes
1.  Fork the repository.
2.  Create a new branch for your feature (`git checkout -b feature/amazing-feature`).
3.  Commit your changes (`git commit -m 'feat: add amazing feature'`).
4.  Push to the branch (`git push origin feature/amazing-feature`).
5.  Open a Pull Request.

## Development Setup
1.  Clone the repo.
2.  Install dependencies: `pip install -r requirements.txt`.
3.  Create a `.env` file with your API keys (if applicable).
4.  Run the app: `./start.sh`.

## Code Style
- Use Python type hints.
- Keep functions small and focused.
- Explain complex logic with comments.
